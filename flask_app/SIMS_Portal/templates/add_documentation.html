Adding Documentation

  • Draft Documentation in WordPress

    The guides, tutorials, and documentation for the SIMS network are drafted in WordPress and hosted at learn-sims.org. This allows for better content management than the Flask backend that powers the SIMS Portal. If you don't have an account with the learning site and would like to contribute, reach out to a Portal admin to get you set up.

  • Article Name and Summary

    Use the same name as the documentation on WordPress. The summary is a brief description of the document, which should also be the same as what you wrote on WordPress in the "Excerpt" box.

  • Category

    All guides should be able to fit into the categories we use for remote support or guidance around design. If your documentation doesn't fit these categories, reach out to an admin.
